    TO INSTALL NEST GLOBALLY:
        npm install -g @nestjs/cli

    TO CREATE NEW NEST PROJECT:
        nest new <PROJECT NAME>

    TO CREATE NEW MODUILE:
        nest g mo <SPECIFY THE PATH WHERE TO CREATE>

    TO CREATE NEW CONTROLLER:
        nest g co <SPECIFY THE PATH WHERE TO CREATE> <SPECIFY THE CONTROLLER NAME> --no-spec "TO AVOID THE TEST FILES"
